Default Commands
  • >ping - You can check bot current connection
  • >help - Give commands how to use a bot
Music
  • >p - Search YouTube to find and play the music that you type
  • >q - You can see the music that I have reserved so far
  • >s - You can vote current play music to skip
DJ
  • >fs - Force the currently playing music to be skipped / Server management only
  • >volume [0 ~ 200] - Adjust the volume of music / Server management only
Admin
  • >setdj [Tags] - This is the function to point as a DJ for the bot. / Server management only
    Owner
    • >stat [text] - Change the state of the game you play on the Bot / Bot Manager Only
